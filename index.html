DATA STRUCTURES AND ALGORITHMS

Unit 1: Data Structures Overview and Arrays

  • Data Types
    • Defines Type of values that can be stored in a variable.
    • Can be of types such as Integer,Float,Double,Char and Many Other.
      • Example:- int var = 12;
      where var is a variable of type int and 12 is the value of the variable.
    • They can be either Fundamental(Primitive) or Derived(non-primitive).
  • Fundamental and Derived Data Type
    • Fundamental Data Types are the basic data types provided by a programming language.
    • Derived Data Types are built from the fundamental data types.
  • Abstract Datatype
    • Hides Implementation details.
    • Conceptual model.
    • Defines set of operations and Behaviour.
    • Explains What without explaining about How.
    • Provides an implementation-independent view,Hence called Abstract.
  • Data Structure
    • Fundamental building block of Computer Programming.
    • Organized data Storage and Manipulation within Programs.
    • Efficient access methods(Accessing and Updating),effective algorithms.
    • Processing,Retrieving and Storing data.
    • Supports Operations.
    • Data Structure can either be Linear(Array/Linked list/Queue/Stack) or Non-linear(Binary Tree/Graph).
    • Classification : -
  • Importance of Data Structures
    • Efficient data handling.
    • Faster algorithms.
    • Optimized memory use.
    • Better problem solving.
    • Foundation for programming.
    • Data Structure can either be Linear(Array/Linked list/Queue/Stack) or Non-linear(Binary Tree/Graph).
  • Array as Data Structures (Characteristics,Advantages,Disadvantages)
    • Linear Data Structure.
    • Stores multiple Values.
    • Same data type of all values.
    • Access via index.
    • Starts at index 0.
    • Can either be Single dimensional or Multidimensional
    • Example
      • int arr[5] = {10, 20, 30, 40, 50};
      // Lower Bound: 0,first element and index 0
      // Upper Bound: 4,last index and size 1

    • Disadvantages
  • Representation of Array - Single and Multidimensional.
    • Linear Data Structure.
    • Elements in one line.
    • Excessed with one index.
      • int arr[5] = {10, 20, 30, 40, 50};
      // Indexes: 0 1 2 3 4
    • Array of Arrays.
    • Common 2D[Matrix]
    • Accessed with two indices.
      • int mat[2][3] = { {1, 2, 3}, {4, 5, 6} };
      // Rows: 2, Columns: 3
